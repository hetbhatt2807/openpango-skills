## 📌 Description
To support Phase 4 (Monetization & Marketplace), we need a backend API that acts as a registry for OpenPango skills. Developers should be able to publish skills, and users should be able to download them via the CLI.

### ✅ Requirements
- Build a lightweight Node.js/Express or Python/FastAPI backend to serve as the registry.
- Implement endpoints for: Publishing a skill (with a zipped bundle), searching skills, and downloading skills.
- Update the `openpango install <skill>` CLI command to fetch from this remote registry if the skill is not found locally.
- Implement basic versioning and dependency resolution (e.g., Skill A requires Skill B).
