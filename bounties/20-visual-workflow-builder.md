## 📌 Description
To make OpenPango accessible to non-developers, we want to build a "Visual Workflow Builder" in our Next.js website. This will allow users to drag-and-drop skills (e.g., Browser -> Researcher -> Data Analysis -> Email) to create custom agent pipelines without writing code.

### ✅ Requirements
- Implement a node-based drag-and-drop UI using a library like `React Flow`.
- Create visual nodes for all standard OpenPango skills.
- Allow users to configure inputs/outputs for each node (e.g., connecting the output of the 'Researcher' to the input of the 'Email' skill).
- Export the visual graph into a valid JSON/YAML configuration file that the OpenPango `router.py` can parse and execute.
