## 📌 Description
Agents are often blind to visual data. We need to implement a `computer-vision` skill that allows the agent to analyze images, charts, and PDF documents natively.

### ✅ Requirements
- Create `skills/vision/image_analyzer.py`.
- Integrate support for vision-capable models (e.g., GPT-4o vision or local Llava).
- Implement a `describe_image(file_path, prompt)` tool that base64-encodes the image and routes it to the configured vision model.
- Add support for basic OCR (Optical Character Recognition) using a local library like Tesseract for faster text extraction without hitting cloud APIs.
